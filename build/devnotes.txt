to check if http-server is installed:
$ http-server

to install http-server:
$ npm i http-server -g

to run server:
$ npm run serve

to install workbox:
$ npm install workbox-cli -g

to view workbox directory for setup:
$ workbox

to start workbox setup:
$ workbox wizard

to create sw files:
$ workbox generateSW
